What is JavaScript?

JavaScript is a front-end scripting language developed by Netscape that defines the dynamic behavior of web pages. It adds programming logic that enables interaction with users, handles events, and creates responsive web experiences. Simple yet flexible, JavaScript is powerful enough to manipulate the DOM (Document Object Model), allowing developers to dynamically update content and layout without reloading the page. It plays a vital role in making websites interactive, from button clicks to real-time updates. Learning JavaScript is essential for all web developers, as it programs the behavior and enhances the functionality of modern websites.

JavaScript Syntax Rules

JavaScript syntax rules define how code must be written to be understood and executed by the browser. These rules include using semicolons to end statements, proper use of brackets {} for blocks, and parentheses () for function calls or conditionals. Variables must be declared with let, const, or var, and strings must be enclosed in quotes.

Semicolons are optional

In JavaScript, semicolons (;) are not strictly required at the end of every statement because the JavaScript engine uses a feature called Automatic Semicolon Insertion (ASI) to interpret line endings.However, relying on ASI can sometimes lead to unexpected bugs or behavior, especially when writing complex code or multiple statements on one line.

Let the code wrap (Avoid line breaks in the middle of a statement)

JavaScript typically expects a complete instruction on one line or properly structured with correct continuation. Inserting line breaks in the middle of a statement (for example, breaking a function call, expression, or assignment in an unexpected place) can confuse the JavaScript interpreter and cause syntax errors or logic bugs. To maintain readability and functionality, ensure statements are written clearly and wrapped only when necessary and valid.

JavaScript is case sensitive

JavaScript treats uppercase and lowercase letters as distinct, meaning identifiers such as variable names, function names, and object properties must match exactly. For example, myVar, MyVar, and myvar would be interpreted as three separate variables. Developers must be consistent with naming conventions and capitalization to avoid referencing undefined variables or functions.

JavaScript Variables

JavaScript variables are temporary storage container for data and can be declared using the three keywords that consists of different behaviors.

VAR

The var keyword is the oldest way to declare variables in JavaScript and has function-level scope. It can be re-declared and updated, but it may cause confusion due to hoisting behavior.

LET

The let keyword was introduced in ES6 and provides block-level scope, making it more predictable than var. It allows variable values to be updated but not re-declared within the same scope.

CONST

The const keyword is used to declare variables with values that should not change once assigned. It has block-level scope, and while the variable itself can't be reassigned, the contents of objects can still be modified.

JavaScript Data Types

Primitive Datatypes in JavaScript are basic data types that store a single value directly in a variable. They are immutable, meaning their actual value cannot be changed once created. There are five main types of primitive datatypes:

Number

It represents numeric values, including integers and floating-point numbers.

String

It is used to represent text enclosed in single, double, or backtick quotes.

Boolean

It holds either true or false, often used in conditional logic.

Undefined

It is a variable that has been declared but not assigned a value.

Null

It represents the intentional absence of any value or object.

JavaScript Arithmetic Operators

Arithmetic Operators in JavaScript are used to perform mathematical calculations between numeric values or expressions. These operators include addition, subtraction, multiplication, division, and more. They can also be used with strings for concatenation (in the case of the + operator). Below are descriptions and examples of the common arithmetic operators:

Addition

Performs addition between numbers or concatenates strings.

Subtraction

Subtracts the right-hand operand from the left-hand operand.

Multiplication

Multiplies two numbers.

Division

Divides the left-hand operand by the right-hand operand.

JavaScript Standard Pop-Up Boxes

Standard Pop-up Boxes in JavaScript are built-in dialog boxes used to interact with users. These include alert, confirm, and prompt boxes—each serving a different purpose for user interaction.

Alert Box

This displays a simple message with an OK button. It’s useful for showing warnings or information to the user. Popping out above, it shows a notification.

Confirmation Box

Shows a message with OK and Cancel buttons, returning true if OK is pressed, and false otherwise. It’s commonly used for asking the user to confirm an action.

Prompt Box

Displays a dialog with a message and an input field for the user to enter a value. It returns the entered value or null if canceled.

JavaScript Statements

JavaScript Statements are instructions that the browser reads and executes in sequence to perform specific actions. They can include variable declarations, assignments, function calls, or control flow structures like conditionals and loops. Each statement typically ends with a semicolon, and they form the backbone of any JavaScript program.

Declaration and Assignment

Declaration and Assignment statements can be combined to create and initialize a variable in a single line. This makes the code concise and clear, allowing you to define a variable and assign it a value at once.

Conditional Statement

Conditional Statements are the ones that allows decision-making in your code based on conditions. Executes different blocks of code depending on whether a condition is true or false.
