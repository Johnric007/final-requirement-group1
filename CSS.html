Definitions of CSS

CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance and layout of HTML or XML documents. It allows developers to separate content from design, making it easier to apply consistent styles—such as colors, fonts, spacing, and positioning—across multiple web pages.

History of CSS

Date/Year Version / Milestone Description
1996 CSS1 introduced Created by Håkon Wium Lie and Bert Bos; Focused on basic styling: fonts, colors, margins, padding, and text alignment.
1998 CSS2 released Expanded styling capabilities with absolute/relative positioning, z-index, media types, and bidirectional text support.
2005-2011 CSS2.1 becomes recommended Refined and corrected CSS2. Became the stable base for browser support and current layout behavior.
2009-Present CSS3 CSS3 introduced as modular specifications, allowing faster and independent development of features like transitions, animations, gradients, flexbox, and media queries.
2012–2017 Widespread adoption of CSS3 features Modern browsers start supporting features like border-radius, box-shadow, flexbox, keyframes animations, and :nth-child() selectors.
2018-Present Advanced layout models and modern features CSS Grid Layout becomes widely supported, offering powerful two-dimensional layout control. Variables (--custom-properties), clamp(), aspect-ratio, and container queries enhance responsiveness and flexibility.

What is CSS Selectors?

CSS selectors are patterns used in stylesheets to select and style specific HTML elements. They allow you to apply styles (like color, size, font, spacing) to elements based on their tag name, class, ID, attributes, or position in the HTML structure. Selectors are the core part of how CSS knows what to style on a webpage.

CSS Background and Colors

The CSS Background and Colors properties let you set the text color, background color, and even background images of elements.

CSS Background

The CSS background property allows you to control the visual background of an element, such as its color, image, position, size, and repeat behavior. It plays a key role in defining the look and feel of a webpage, helping to add contrast, texture, or visual interest to elements like headers, sections, buttons, and entire layouts. CSS provides both shorthand and individual background properties to give designers precise control over how backgrounds behave.

CSS Color

The CSS color property is used to set the text color of HTML elements. It plays a vital role in web design by enhancing readability, creating visual contrast, and matching the overall aesthetic of a website. CSS allows developers to specify colors in multiple formats, including named colors (like red or blue), hex codes (e.g., #ff0000), RGB values (e.g., rgb(255, 0, 0)), HSL values, and even transparent colors using RGBA or HSLA.

CSS Fonts and Styling

The CSS Font and Text Styling in CSS enhance the appearance and readability of web content by customizing typography and text layout.

CSS Fonts Styling

Font styling in CSS refers to the customization of text appearance by modifying the font family, style, weight, size, and variant. Common properties include:

  • font-family – sets the typeface (e.g., Arial, Georgia, sans-serif).
  • font-size – controls the size of the text (e.g., 16px, 1.2em).
  • font-style – adds styles like italic (e.g., normal, italic).
  • font-weight – changes the boldness (e.g., normal, bold, 400, 700).
  • font-variant – enables small-caps and other text effects.

CSS Text Styling

Text styling in CSS focuses on the layout and presentation of text beyond the font itself. It includes properties like:.

  • text-align – aligns text left, right, center, or justify.
    • text-decoration - – adds underlines, overlines, or line-through (used for links, emphasis, or deletion).
  • text-transform – changes capitalization (e.g., uppercase, lowercase, capitalize).
  • letter-spacing – adjusts spacing between letters.
  • line-height – sets vertical space between lines of text.

What is a Box Model?

The CSS box model is a fundamental concept that describes how elements are structured and spaced on a webpage. Every HTML element is considered a rectangular box made up of four parts: content, padding, border, and margin. The content holds the actual text or media, padding adds space inside the border, the border wraps the element, and the margin creates space between elements—understanding this model is essential for controlling layout and spacing in web design. It allows developers to fine-tune how elements interact with one another, ensuring proper alignment and preventing overlap or unexpected gaps. Mastering the box model is key to creating clean, responsive, and visually balanced web layouts.

CSS Positioning and Layout

Positioning and Layout in CSS control how elements are arranged and displayed within a webpage. It uses different position types such as static, relative, absolute, fixed, and sticky to determine an element's placement in relation to the document flow or other elements. With these properties, developers can layer elements, create overlapping content, or keep elements fixed during scrolling. Layout techniques like Flexbox and Grid also play a crucial role in aligning and distributing space efficiently across complex designs. Understanding positioning and layout is essential for building responsive, organized, and visually appealing web pages.

Here are some common Position Types:

Static Position

Static-positioned elements are set as the default and have no impact on right, left, bottom, or top properties. They don’t have any special direction. It was made positioned on the normal Page.

Relative Position

It is helpful while creating a CSS layout, and it finds quite difficult. An HTML element with this property works the same as a static, and they differed by setting the right and left values to the ancestor element.

Absolute Position

In this position, an element places itself with respect to itself and excludes itself from the document’s normal flow. Generally, they position themselves in the top-left of their ancestor or parent element.

CSS Display and Visibility

CSS Display and Visibility control how elements appear and occupy space on a webpage. The display property determines the layout behavior of an element, such as block, inline, flex, or grid, affecting how elements are arranged in relation to others. On the other hand, the visibility property controls whether an element is visible or hidden, without affecting the layout—using values like visible or hidden. Unlike display: none, which removes the element from the layout entirely, visibility: hidden keeps the space reserved. These properties are essential for dynamic interfaces, responsive design, and managing user interactions smoothly.
